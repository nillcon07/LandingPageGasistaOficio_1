# Landing Page — Gasista Matriculado CABA/GBA

Landing page profesional para servicio de gasista matriculado. Diseño premium con estética original (dark/naranja), completamente responsiva y optimizada para conversiones.

## Estructura del proyecto

```
/
├── index.html              # Página principal
├── assets/
│   ├── css/
│   │   └── style.css       # Estilos y animaciones
│   └── js/
│       └── script.js       # Interactividad (FAQ, scroll, navbar)
└── README.md
```

## Características

- ✅ Hero con estadísticas animadas y CTA doble (WhatsApp + Llamar)
- ✅ Barra de urgencia para cortes de gas
- ✅ 4 tarjetas de servicios con hover effects
- ✅ Sección "¿Por qué elegirnos?" con card visual
- ✅ Proceso paso a paso (4 etapas)
- ✅ Cobertura por zonas (CABA + GBA Norte/Oeste/Sur)
- ✅ FAQ accordion interactivo
- ✅ CTA de contacto final con glow animado
- ✅ Footer completo
- ✅ Botón flotante de WhatsApp
- ✅ Scroll reveal animations
- ✅ 100% responsivo (mobile-first)

## Personalización

Antes de publicar, reemplazar en `index.html`:
- `+5491100000000` → número de teléfono real
- `GasExpert` → nombre real del profesional (opcional)

##  Deploy

Abrir `index.html` en cualquier navegador. Para publicar online: GitHub Pages, Netlify o cualquier hosting estático.
